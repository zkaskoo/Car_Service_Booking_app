
### Code Example

```php
class BookingAvailabilityService
{
    public function getAvailableSlots(Carbon $date, array $serviceIds): array
    {
        // 1. Get working hours
        $workingHours = WorkingHours::where('day_of_week', $date->dayOfWeek)->first();
        if (!$workingHours || $workingHours->is_closed) {
            return [];
        }

        // 2. Check blocked dates
        if ($this->isDateBlocked($date)) {
            return [];
        }

        // 3. Calculate duration
        $totalDuration = Service::whereIn('id', $serviceIds)->sum('duration_minutes');

        // 4. Get service bays
        $serviceBays = ServiceBay::where('is_active', true)->get();

        // 5. Get existing bookings
        $existingBookings = Booking::whereDate('booking_date', $date)
            ->whereNotIn('status', ['cancelled', 'no_show'])
            ->get();

        // 6. Generate available slots
        return $this->generateSlots(
            $workingHours,
            $totalDuration,
            $serviceBays,
            $existingBookings
        );
    }
}
```

### Status Definitions

| Status | Description |
|--------|-------------|
| `pending` | Booking created, awaiting confirmation |
| `confirmed` | Admin/system confirmed the booking |
| `in_progress` | Service has started |
| `completed` | Service finished successfully |
| `cancelled` | Booking cancelled by user or admin |
| `no_show` | Customer didn't show up |
| `rescheduled` | Booking moved to different date/time |
